# mp-cz-cli
小程序-脚手架
## 🎯 Install

```
npm i -g mp-cz-cli
```

## 🚀 Usage

Create a new project with the following instructions:

```
mp-cz init
```

Go to the created directory and start webpack devServer via npm or yarn:

```
cd [name]
npm install
```
